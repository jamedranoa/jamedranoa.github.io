Roof

LiveGitHub

Built using Ruby on Rails, uses Backbone.js views which are appended in different elements of a classic rails view. Uses Mapbox.js and Geocoder to localize a room, makes a search verifying location and dates availability, The User can make a request on the fly using jQuery UJS, When an user accepts a request, automatically reserve the dates of the room and eliminate any overlapping request, own design and style made by "hand" using only CSS.

Features

  • Users can:
    • search rooms by location and date.
    • manage their profiles.
    • list and edit rooms.
    • request the use of another user's listed room.
    • approve or deny requests of their own listed rooms.
  • Dates available are updated every time a request is approved.
  • Incompatible requests are destroyed at the moment of approval.
  • Places and users are reviewable.
  • Users receive notifications of requests, reviews, approvals, and rejections.

Technologies

  • Photo saving and resizing using Paperclip and ImageMagick.
  • Address location using geocoder.
  • Location display using Mapbox's Javascript API.
  • Backbone.js views mixed into classic rails views.
  • Polymorphic review model for users and places.
  • Polymorphic notification model.
  • Uses jQuery UJS for creating reviews, requests and notificiations on the fly.
  • Pagination using Kaminari
